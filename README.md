# Meteoric Garden Shop

## Description

Meteoric Garden Shop is a test project where I designed a wireframe in Figma and then implemented it into actual working code using the Meteor framework. I chose Meteor somewhat arbitrarily simply to preserve and build upon my design work.

## Installation

Follow these steps to run the project locally:

1. Install Meteor
   ```bash
   npx meteor
   ```

2. Clone the repository:
   ```bash
   git clone https://github.com/stephen-cpe/meteoric-garden-shop-v4.git
   cd meteoric-garden-shop-v4
   ```

3. Install dependencies:
   ```bash
   meteor npm install
   ```

4. Start the development server:
   ```bash
   meteor npm start
   # or
   meteor run
   ```

5. Open your browser and navigate to `http://localhost:3000`. Or `http://localhost:3000/admin` to access admin page.
